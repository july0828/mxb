国内现状

慢性病

减轻慢病负担，提高生活质量，让健康成为我们每一天的习惯。

在中国大陆，慢性病已成为公共卫生领域的重要挑战。随着人口老龄化、生活方式改变和环境因素等因素的影响，慢性病的发病率不断上升。 最常见的慢性病包括高血压、糖尿病、冠心病、慢性阻塞性肺疾病（COPD）、肥胖症等。据统计，中国大陆有超过3亿人患有高血压，超过1亿人患有糖尿病。 慢性病对人们的健康和生活质量造成了严重影响，同时也给社会经济带来了巨大负担。根据中国疾病预防控制中心的数据，慢性病已成为中国大陆最主要的死亡原因之一。 中国政府对慢性病的防控给予了重视。在过去几年中，中国大陆采取了一系列措施，包括推广健康教育、加强慢性病筛查和管理、提高医疗服务水平等，以应对慢性病的挑战。
